#################################################
ONTAP (AFF/FAS/Select/Cloud/Amazon FSx for ONTAP)
#################################################

This section of the documentation tells you all you need to know on
creating an ONTAP backend. These instructions apply for all ONTAP
clusters no matter where you choose to run it; on dedicated hardware
(AFF/FAS), on commodity hardware (Select) or on a hyperscaler (Cloud).

Begin by understanding what the different drivers have to offer in the
:ref:`ONTAP Storage Driver <Choosing a driver>` section.
